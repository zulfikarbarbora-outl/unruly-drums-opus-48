# unruly-drums-opus-48

Opus 48 kbps preview MP3s for the **karoryfer.unruly-drums** sample library.

## About

This repo holds **48 kbps Opus previews** for fast streaming in the web DAW.
Opus at 48k delivers near-CD quality at ~35% the size of MP3 128k.

- **Files**: 1736 Opus previews
- **Bitrate**: 48 kbps VBR
- **Container**: Ogg Opus (.opus)
- **Source**: [`zulfikarbarbora-outl/karoryfer.unruly-drums`](https://github.com/zulfikarbarbora-outl/karoryfer.unruly-drums)
- **Branch**: `main`

## URL pattern

```
https://raw.githubusercontent.com/zulfikarbarbora-outl/unruly-drums-opus-48/main/<filename>.opus
```
